# 🏥 Clinic Data Analysis Project

## 📌 Overview

This project analyzes clinic appointment data to uncover trends in patient visits, revenue, and operational performance.

## 🧹 Data Cleaning

* Standardized gender values
* Converted date columns into proper format
* Cleaned billing amounts (removed currency symbols)
* Handled missing values

## 📊 Analysis & Visualization

Using Excel Pivot Tables and charts:

* Revenue trends over time
* Department workload analysis
* Doctor performance
* Follow-up rate distribution

## 🔍 Key Insights

* Some departments handle significantly more patients
* Revenue varies across time periods
* Follow-up rates reveal patient retention patterns

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Data Cleaning Techniques

## 📁 Project Structure

* `data/` → cleaned dataset
* `dashboard/` → Excel dashboard
* `images/` → dashboard screenshots
